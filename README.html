PostApp

Aplicacion en AngularJs

Esta aplicacion es de prueba para ver como funciona Angularjs consumiendo API'S REST

Utilizamos una API fake: Jsonplaceholder Esta API nos permite simular los metodos para hacer peticiones a un servidor tales como:

  • POST -> Para agregar un nuevo recurso
  • GET -> Para consultar un recurso en especifico
  • PUT -> Para actualizar un recurso
  • DELETE -> Para eliminar un recurso espeficio

Se utilizaron los siguientes componentes de angular:

  • AngularRoute
  • AngularResource
  • AngularLocation

Esta aplicacion de prueba te permite crea, consultar, actualizar y eliminar algun post que estan en la API de json placeholder.
